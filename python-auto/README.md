# 自动化任务处理系统

## 功能说明

这个系统可以自动从数据库读取待办任务，并使用 Cursor AI 自动生成项目。

## 工作流程

1. **连接数据库**：从 `user_task` 表中查询所有 `task_status = 'pending'` 的任务
2. **任务队列**：将待办任务按创建时间排序，放入队列
3. **自动处理**：逐个处理队列中的任务
   - 点击 Cursor 输入框
   - 生成并输入提示词
   - 点击发送按钮
   - 等待 AI 生成项目
   - 点击 Accept 按钮
   - 更新任务状态

## 提示词格式

系统会根据任务信息自动生成提示词：

```
创建一个名字为{user_id}文件夹，写一个规模为{task_type}的项目，技术选型为{task_technology}，项目描述为{task_description}，不要乱写文档，只需要最后写一个运行文档就可以，然后将这个项目压缩到这个项目的根目录中
```

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. **确保数据库配置正确**：编辑 `db_connector.py` 中的 `DB_CONFIG`

2. **确保 Cursor 已打开**：AI 输入框、发送按钮、Accept 按钮都可见

3. **运行脚本**：
```bash
python auto.py
```

## 任务状态

- `pending`: 待办
- `processing`: 处理中
- `completed`: 已完成
- `failed`: 失败

## 等待时间

当前所有项目的默认等待时间为 **45 分钟（约 2700 秒）**，用于保证在有详细规划文档的情况下，AI 有足够时间完成项目生成和代码编写。

## 注意事项

1. 运行前确保 Cursor 窗口可见且输入框、按钮都在屏幕上
2. 可以将鼠标移到屏幕角落强制中止脚本（FAILSAFE）
3. 任务之间会有3秒间隔，避免操作过快
4. 如果某个步骤失败，任务会被标记为 `failed`，继续处理下一个任务
